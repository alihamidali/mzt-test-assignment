# mzt-test-assignment

#Prerequisites

```
PHP 8.0
npm
mysql
```

# Project Setup

```
$ git clone git@github.com:alihamidali/mzt-test-assignment.git

$ npm install

$ composer install

$ cp .env.example .env // update db configurations and smtp configurations

$ php artisan migrate

$ php artisan db:seed

$ php artisan serve
```
